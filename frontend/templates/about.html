About HepatoAI

A final-year B.Tech project in AI-assisted liver disease detection

Project Overview

HepatoAI is a complete end-to-end Multi-Modal Deep Learning system designed to assist in early liver disease detection. It fuses two distinct data modalities — clinical tabular data (lab results) and liver ultrasound images — using a dual neural network architecture to improve diagnostic accuracy beyond single-modality approaches.

The system classifies patients into one of three categories: Normal, Fatty Liver, or Cirrhosis, with accompanying probability scores and AI-generated explanations using SHAP and Grad-CAM.

Explainable AI (XAI)

SHAP (SHapley Additive exPlanations): Used with the ANN model to explain which clinical features most influenced the prediction. SHAP values are computed using DeepExplainer/GradientExplainer and displayed as force plots and bar charts.

Grad-CAM (Gradient-weighted Class Activation Maps): Applied to the CNN model to generate heatmaps showing which regions of the ultrasound image were most important for classification. These heatmaps are overlaid on the original image for visual interpretation.
